使用阿里云搭建Minecraft服务器

发表于 2016-03-07   |   分类于 服务器   |  

前两天，由于使用了github搭建了我的blog，之前买的阿里云服务器（云翼计划，学生党10块一个月）就闲置了，自己感觉好浪费，闲来无事发现同学都喜欢玩我的世界，就自己动手搭建了一台我的世界服务器，24小时开机，满足我的同学们一起联机的愿望。
搭建我的世界服务器非常简单，只需要简单的几个步骤。

我买的服务器是阿里云ecs，配置为最低的，20gb硬盘，系统为Ubuntu14.04.
先用putty连上服务器后，安装java和screen，screen是用来防止关闭putty后服务器脚本被挂起。

sudo apt-get update
sudo apt-get install openjdk-6-jre-headless
sudo apt-get screen

然后检查java安装是否成功。

java -v

（也可以去java官网下载rpm包然后上传至服务器进行安装）

安装完java后，我们要去下载cauldron（原来叫MCPC）
MCBBS的地址
选择一个版本，并下载对应的librarys，解压librarys，将里边的三个文件和cauldron一起上传到服务器，然后新建个shell脚本，起名为“start.sh”，里边写入这一句：

java -Xincgc -Xms256m -Xmx750m -Xss1M -server -jar cauldron-1.7.10-*.jar

保存，更改权限

chmod a+x start.sh

运行脚本，第一次可能会安装失败，需要打开eula.txt，将其中的false改为true。

再次运行脚本，看见“done”，的字样就表示安装成功，输入stop停止服务。
vi打开server.properties，找到online-mode=true,改为false，保存。

screen -S mc

在新的终端里边运行start.sh脚本，至此，一个可以24小时运行的我的世界服务器就搭建完成了。

注意：服务器端与本地端需保持一致，forge也必须存在，如果安装mods的话把mods文件上传到mods文件夹，注意有些可以单机玩耍的mods不能安装到服务器上。
